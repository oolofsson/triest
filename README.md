# Triest
[link to report](https://static.googleusercontent.com/media/research.google.com/sv//pubs/archive/45391.pdf)

## Prerequisites
* Java
* Maven
* Gnuplot

## Compiling
```
./compile.sh
```

## Running
```
./run.sh -a algorithm (base or impr, default: base) -f ./resources/(inputfile) ( default: moreno_health_health)
```

## Plotting
```
./plot plot.txt
```
